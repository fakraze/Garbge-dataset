
# Garbage Dataset

This dataset contains **5 categories** of garbage images for classification tasks:
| Category  | Number of Images |
|-----------|----------------|
| **Cardboard** | 305 |
| **Glass** | 112 |
| **Plastic** | 100 |
| **Metal** | 113 |
| **Paper** | 100 |

Total images: **730**
